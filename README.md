# The Mercator Game 🌍

A geography game that challenges your intuition about map projections.

## How to Play
1. The game presents a world map using an **Oblique Mercator Projection**.
2. A random city is chosen to be the **North Pole** (singularity at the top of the map).
3. The map orientation is randomized (Meridian Vertical or Parallel Vertical).
4. Your goal is to find a specific **Target City** on this distorted map.
5. Click to make your guess. A green dot reveals the true location.

## Technologies
- **D3.js** (v7) for map projection and interaction.
- **TopoJSON** (v3) for map data.
- **HTML/CSS/JS** (Single file implementation).

## Deployment
This game is designed to be hosted on **GitHub Pages**.
Simply push the `index.html` to a GitHub repository and enable Pages in the repository settings.
